# GST Invoice Generator – cognisutra

**Live demo:** [gst-invoice-generator](https://gstinvoicegenerator-one.vercel.app/)

A professional, 100% client-side GST invoice generator. No registration, no backend – all data stays in your browser. Perfect for freelancers, small businesses, and startups.

## ✨ Features

- **10 stunning templates** (Minimal, Branded Red, Navy Pro, Modern Teal, Elegant, Slate, Vibrant, Trust Green, Luxury, Compact Mono)
- **Logo upload** (PNG/JPG/SVG, max 2MB)
- **Digital signature** (draw or upload image)
- **UPI QR code** – automatically generated from UPI ID
- **GST compliant** – CGST/SGST (intra-state) & IGST (inter-state)
- **Line items** with HSN/SAC, quantity, discount, GST rates (0%,5%,12%,18%,28%)
- **Automatic tax calculations** & amount in words
- **PDF export** (print-ready) + instant print
- **Auto-save** – draft is saved automatically, reload won't lose data
- **Full preview** – see the styled invoice before exporting
- **Responsive** – works on desktop, tablet, and mobile
- **No signup, no tracking, no server** – fully private

## 🚀 How to use

1. **Clone or download** this repository.
2. Open `index.html` in any modern browser (Chrome, Firefox, Edge, Safari).
3. Fill in your business & client details.
4. Add line items, choose a template, upload logo/signature.
5. Click **Export PDF** or **Print**.
6. Use **Save draft / Load draft** to keep multiple versions.

## 🛠️ Tech stack

- HTML5 / CSS3 (flex, grid, custom properties)
- Vanilla JavaScript (no frameworks)
- Google Fonts (Inter, Playfair Display, JetBrains Mono)
